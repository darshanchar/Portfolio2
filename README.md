# 🌐 Darshan Achar – Personal Portfolio

This project is a **Personal Portfolio Website** built using **HTML5 and CSS3**.
It showcases my profile, skills, and contact information as I continue learning **Full Stack Web Development**.

The portfolio includes a **modern navigation bar, hero section, animated logo text, social media icons, and profile image styling**.

---

## 🚀 Live Demo

View the live website hosted using **GitHub Pages**:

```
https://darshanchar.github.io/Portfolio/
```

---

## 📌 Features

* Responsive **navigation bar**
* **Hero section** with introduction
* **Animated glowing logo text**
* Social media icons using **Font Awesome**
* Stylish **profile image with glow effect**
* Clean and simple **UI design**
* Hover effects for buttons and menu items

## 🛠️ Technologies Used

* **HTML5**

```

```

3. Run the website by opening:

```
index.html
```

in your browser.
